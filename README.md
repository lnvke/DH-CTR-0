# CTR-0
Cheaper Than aRduino is a custom devboard meant to be a cost-effective alternative my choice of the Arduino Nano series of devboard hobbyist microcontrollers. It is based of the same RP2040 microcontroller and will provide similar performance compared to its more costly counterpart, the Arduino Nano V3, which is the board I usually use for projects that require a microcontroller.
